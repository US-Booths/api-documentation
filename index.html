Authentication

The API uses Laravel Sanctum for token-based authentication.

POST /api/login Public
Get authentication token

Request

{ "email": "user@example.com", "password": "your_password" }

Response

{ "success": true, "token": "1|abcdef123456...", "user": { "id": 1, "name": "User", "email": "user@example.com" } }

Using the Token

Authorization: Bearer 1|abcdef123456...

Rate Limits

  • Public: 60 req/min per IP
  • Auth: 10 req/min per IP (login/register)
  • Admin: 600 req/min per user
  • Client: 120 req/min per IP
  • Webhooks: 120 req/min per IP
